# bagpipe-devtools
Crawl data from website extension using developer tool.
## Install
- Run `cd bagpipe-v2`.
- Run `npm build`.
- Add `build` folder as an Extension of Google Chrome.
- Run `npm install`.
- Run `npm start`.
- Run `cd ../server`
- Run `npm install`.
- Run `npm run dev` in `server` folder.
- Open a website.
- Open Developer tool (F12).
- Navigate to `bagpipe-devtool` panel.
